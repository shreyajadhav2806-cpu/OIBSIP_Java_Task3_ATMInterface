# ATM Interface

## 📌 Objective

The objective of this project is to develop a console-based ATM Interface using Core Java. The application simulates basic ATM operations such as user authentication, balance management, cash withdrawal, deposit, fund transfer, and transaction history using Object-Oriented Programming concepts.

---

## ✨ Features

* Secure User Login
* Deposit Money
* Withdraw Money
* Transfer Funds
* View Transaction History
* Balance Management
* Menu-Driven Console Application
* Object-Oriented Programming Implementation

---

## 🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Scanner Class
* ArrayList
* Methods & Constructors

## ▶️ How to Run

1. Open the project folder.
2. Compile all Java files:

```bash
javac *.java
```

3. Run the application:

```bash
java ATMInterface
```
